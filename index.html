grandrock by acornejo

GTK RandR interface

GRandRock is a rudimenentary (yet useful) tool to change resolutions and manage multiple screens using the X RandR extensions (through xrandr).

Try it out!

Dependencies

The only dependency of grandrockr is x11-server-utils (for the xrandr command line tool), and either gtk-2.0 or gtk-3.0.

To compile grandrockr from source, these are the recommended Debian/Ubuntu packages:

libgtk-2-dev | libgtk-3-dev, autotools-dev, dh-autoreconf, debhelper, x11-server-utils

Install

Basic installation: 
git clone git://github.com/acornejo/grandrock.git
cd grandrock
./autogen.sh --prefix=/usr
make
sudo make install
